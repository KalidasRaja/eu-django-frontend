Main Features

Individual Unit Alarms

The key to a successful remotely managed self storage facility lies in the security of each individual Storage unit. We are able to integrate any and all existing solutions for individual unit alarm, from wired to wireless door magnets, but the most complete security available is provided by our PIR/microwave dual technology sensors to detect movement intelligently within each unit while preventing false alarms.

  • All sensors are installed on a single BUS cable that runs through the facility. Easy and quick installation
  • Remotely adjust sensitivity of each individual sensor even while units are rented
  • Anti-masking feature on each sensor to protect against more sophisticated burglar attempts

Electric Locks On Units

6Security can manage any electric lock solution on individual units thanks to its sturdy and programmable relays. We narrow our remote controlled electric lock solutions down to two.

  • The tenant access code at main door temporarily activates a button at unit. The tenant walks to his unit, presses a button and the lock is released.
  • When the tenant digits his access code at the main door, the unit lock is unlocked instantly.

6Security does not require complex wiring for adding electric locks. Electric locks relay commands can be placed on the same BUS cable of the alarm sensors for each unit.

New Revenue Streams for your Facility

How about an Amazon delivery box?

It starts with providing to each client a separate "Parcel Box Code"

When making an order on Amazon or any e-commerce site, clients can use your Self Storage address for the delivery and write the Parcel Box Code and unit number on the address field. The courier will come to your facility, insert the code at the gate, go in and deposit the parcel in the Parcel Box that he will open by pressing a button.

Your client gets an email for 6security when the package is delivered to the parcel box. So he or she knows it is ready to be picked up. A tremendous time and hassle saving for your clients. Ask us for more details from our case study in a Self Storage facility in Italy.

How about a Bike Box ?

If your Self Storage is in a very densely populated neighbourhood, you might want to offer folks a unit with a few bike parking spots. Clients get a code which unlocks the unit and they can get their bike 24 hrs a day. You can also set electric locks on each individual bike.
